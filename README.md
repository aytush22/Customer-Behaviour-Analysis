# Customer Trends Data Analysis

## About This Project

In this project, I worked on building a **complete end-to-end data analysis pipeline** — starting from raw data all the way to business insights and visualization.

The idea was to simulate how data is actually used in a real company to understand **customer behavior and buying patterns**.

---

## What I Did

### Data Cleaning & Exploration (Python)

I started with raw data and:

- Cleaned missing and inconsistent values
- Explored patterns using visualizations
- Prepared the dataset for further analysis

---

### Data Analysis (SQL)

Once the data was ready, I moved it into a database and:

- Wrote SQL queries to answer business questions
- Analyzed customer segments and behavior
- Identified trends in purchasing patterns

---

### Dashboard Creation (Power BI)

To make the insights easy to understand:

- Built an interactive dashboard
- Highlighted key metrics and trends
- Made the data more accessible for decision-making

---

### Reporting & Presentation

Finally:

- Summarized key insights
- Converted analysis into business recommendations
- Created a presentation to communicate findings clearly

---
